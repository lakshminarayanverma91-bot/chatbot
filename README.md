# 🤖 Simple NLP Chatbot

A beginner-friendly NLP based chatbot built using Python and NLTK. This
chatbot can understand basic user inputs and respond in English or
Hinglish.

------------------------------------------------------------------------

## 📌 Project Overview

This project is a simple Natural Language Processing (NLP) chatbot
created as part of a learning task.

The chatbot: - Takes user input - Processes text using NLP techniques -
Matches it with predefined intents - Returns an appropriate response

It is designed to be easy to understand and beginner-friendly.

------------------------------------------------------------------------

## 🚀 Features

-   💬 Interactive chatbot interface (Terminal-based)
-   🌐 Supports English and Hinglish responses
-   🔀 Random response selection from multiple replies
-   🧠 Basic NLP preprocessing (tokenization, stopword removal)
-   📁 Uses JSON file for storing intents and responses
-   🔄 Easy to modify and expand
-   🎓 Beginner-friendly code structure

------------------------------------------------------------------------

## 🛠️ Technologies Used

-   Python 3
-   NLTK (Natural Language Toolkit)
-   JSON

## 🧠 How It Works

1.  User enters a message
2.  Text is tokenized using NLTK
3.  Stopwords are removed
4.  Input is matched with predefined patterns
5.  A random response is selected
6.  Response is displayed in English or Hinglish
